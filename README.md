# Token UUID

## Description

Make **entity UUID** token available in list of tokens.<return> 
These tokens aren't yet implemented in Token 8.x-1.0 .

## Installing
1. Install the module as normal, note that there is one dependency (token).
2. that's all !  

## Using
* [node:uuid]
* [term:uuid]
* [user:uuid]
* [current-user:uuid]
